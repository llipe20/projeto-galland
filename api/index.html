Finalidade

O projeto consiste em desenvolver uma mini API para fazer a ponte entre um banco de dados e o front end de uma aplicação especifica. Ou seja, a api servirá apenas para essa aplicação, a menos que queira usar pra testar mesmo.

A aplicação especifica, é a construção de um cardápio online para uma sorveteria, onde pretendo usar esta API para coletar os dados de clientes, pedidos e produtos no banco de dados, trazendo mais dinamismo, facilitando o trabalho do desenvolvedor.

Modo de usar

Como havia anteriormente, esta API servirá especificamente para uma única aplicação. Não contém informações relevantes para uso casual e para outros tipos de projetos, igual a API de filmes, satélites e etc...

Mas, caso queira testar e tals, é bem simples. Basta colocar a url da API na sua requisição e configurar os método de envio, cabeçalhos e etc...
